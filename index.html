  • Google Computer Science Summer Institute

    Mountain View, CA | July 2015

    By participating in this program, I was able to learn web app development from Google engineers. Our team project consisted of creating a digital time capsule so users can craft messages that could only be viewed after the desired delivery date.

    purpose

    A time capsule contains artifacts that is buried with the intention of discovering these artifacts in the future. Our team decided to create a digital time capsule called Project M to store messages that will be sent to the desired recipient after a defined amount of time. Essentially, a user can send emails that will be delivered later. Sound like Boomerang? What makes Project M different is that after creating the memory, the user cannot view the information just how a time capsule is not taken out of the ground until the desired date.

    user flow

    view homepage
    sign into gmail
    view profile page that includes
    saved memories chart and create new memory button
    input desired recipent, text, and devliery date with prompted form
    alert with confirmation pop up
    redirect to profile page
